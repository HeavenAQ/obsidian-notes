## Summary

Given a string `s`, return the longest substring of `s` that is a palindrome. If several have the same maximal length, any one of them is accepted.

**Constraints:** `1 <= s.length <= 1000`, `s` consists of digits and English letters.

**Example 1:** `s = "babad"` → `"bab"` (`"aba"` also valid).

**Example 2:** `s = "cbbd"` → `"bb"`.

## Hints

1. A substring `s[i..j]` is a palindrome exactly when `s[i] == s[j]` and the inside `s[i+1..j-1]` is also a palindrome. That is a recurrence on intervals.
2. Fill a boolean table `dp[i][j]` by increasing substring length: length 1 is always true, length 2 needs `s[i] == s[j]`, longer lengths use the recurrence from hint 1.
3. Track the best `(start, length)` while filling the table. (Alternative: expand around each of the `2n-1` centers for O(1) space.)

+++ Solution (click to reveal)

**Approach.** Interval DP on substrings. `dp[i][j]` = true iff `s[i..j]` is a palindrome:

- `dp[i][i] = true`
- `dp[i][i+1] = (s[i] == s[i+1])`
- `dp[i][j] = (s[i] == s[j]) && dp[i+1][j-1]` for `j - i >= 2`

Iterate by increasing length so `dp[i+1][j-1]` is ready before `dp[i][j]`; remember the longest true cell.

**Complexity.** Time `O(n^2)`, space `O(n^2)`. (Expand-around-center gives `O(n^2)` time with `O(1)` space; Manacher's is `O(n)` but overkill here.)

```java
class Solution {
    public String longestPalindrome(String s) {
        int n = s.length();
        boolean[][] dp = new boolean[n][n];
        int start = 0, maxLen = 1;
        for (int i = 0; i < n; i++) {
            dp[i][i] = true;
        }
        for (int len = 2; len <= n; len++) {
            for (int i = 0; i + len - 1 < n; i++) {
                int j = i + len - 1;
                if (s.charAt(i) == s.charAt(j) && (len == 2 || dp[i + 1][j - 1])) {
                    dp[i][j] = true;
                    if (len > maxLen) {
                        start = i;
                        maxLen = len;
                    }
                }
            }
        }
        return s.substring(start, start + maxLen);
    }
}
```

**Why this works.** Every palindrome is either a single character, a matched pair, or a matched pair wrapped around a shorter palindrome — the table enumerates all intervals bottom-up by length, so no case is missed.
