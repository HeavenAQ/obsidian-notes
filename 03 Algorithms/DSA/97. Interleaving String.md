## Summary

Given three strings `s1`, `s2`, and `s3`, determine whether `s3` is formed by an **interleaving** of `s1` and `s2`.

An interleaving of two strings `s` and `t` is a way of splitting `s` into substrings `s1s2...sn` and `t` into substrings `t1t2...tm` (with `|n - m| <= 1`) and concatenating them alternately — either `s1+t1+s2+t2+...` or `t1+s1+t2+s2+...` — to reconstruct the target string, while preserving the relative order of characters from each original string.

**Constraints**

- `0 <= s1.length, s2.length <= 100`
- `0 <= s3.length <= 200`
- `s1`, `s2`, and `s3` consist of lowercase English letters.

**Examples**

1. `s1 = "aabcc"`, `s2 = "dbbca"`, `s3 = "aadbbcbcac"` → `true`
2. `s1 = "aabcc"`, `s2 = "dbbca"`, `s3 = "aadbbbaccc"` → `false`

## Hints

3. If `s1.length() + s2.length() != s3.length()`, it's immediately impossible.
4. Think of it as a 2D grid problem: let `dp[i][j]` mean "can the first `i` characters of `s1` and first `j` characters of `s2` interleave to form the first `i+j` characters of `s3`?"
5. At each cell you only have two choices for where the next character of `s3` came from: the tail of `s1` or the tail of `s2`. Try both and OR the results.
6. Base case: `dp[0][0] = true` (two empty strings interleave to an empty string).

> [!note]+ Solution (click to reveal)
> **Approach**
> 
> This is a classic 2D dynamic programming / string-edit problem. Define `dp[i][j]` as `true` if `s3[0..i+j)` can be formed by interleaving `s1[0..i)` and `s2[0..j)`.
> 
> Transition, for `1 <= i <= m` and `1 <= j <= n` (where `m = s1.length()`, `n = s2.length()`):
> 
> ```javascript
> dp[i][j] = (dp[i-1][j] AND s1[i-1] == s3[i+j-1])
>         OR (dp[i][j-1] AND s2[j-1] == s3[i+j-1])
> ```
> 
> That is, the last character of the interleaved prefix must come from either the end of the `s1` prefix or the end of the `s2` prefix, and whichever one it came from, the rest must already be a valid interleaving.
> 
> Base case: `dp[0][0] = true`. The first row/column reduce to simply matching `s1` (or `s2`) directly against `s3`, since the other string contributes nothing yet.
> 
> If `m + n != s3.length()`, return `false` immediately — a necessary length check that also guarantees the DP indices stay in bounds.
> 
> The answer is `dp[m][n]`.
> 
> **Why this works**: every interleaving of a prefix of `s1` and a prefix of `s2` must end with a character taken from one of the two strings' current prefix ends, so the recurrence exhaustively covers both possibilities and reuses previously verified sub-interleavings — the DP correctly captures all valid splits without re-deriving them.
> 
> **Complexity**: Time `O(m * n)`. Space `O(n)` using a rolling 1D array (shown below); `O(m * n)` if using the full 2D table.
> 
> ```java
> class Solution {
>     public boolean isInterleave(String s1, String s2, String s3) {
>         int m = s1.length(), n = s2.length();
>         if (m + n != s3.length()) {
>             return false;
>         }
> 
>         // dp[j] represents dp[i][j] for the current row i, rolling over i.
>         boolean[] dp = new boolean[n + 1];
>         dp[0] = true;
> 
>         // Initialize first row (i = 0): only s2 contributes.
>         for (int j = 1; j <= n; j++) {
>             dp[j] = dp[j - 1] && s2.charAt(j - 1) == s3.charAt(j - 1);
>         }
> 
>         for (int i = 1; i <= m; i++) {
>             // Update j = 0 column for this row: only s1 contributes.
>             dp[0] = dp[0] && s1.charAt(i - 1) == s3.charAt(i - 1);
> 
>             for (int j = 1; j <= n; j++) {
>                 boolean fromS1 = dp[j] && s1.charAt(i - 1) == s3.charAt(i + j - 1);
>                 boolean fromS2 = dp[j - 1] && s2.charAt(j - 1) == s3.charAt(i + j - 1);
>                 dp[j] = fromS1 || fromS2;
>             }
>         }
> 
>         return dp[n];
>     }
> }
> ```
