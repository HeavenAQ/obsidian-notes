## Summary

Given a string `s` and a dictionary of strings `wordDict`, return `true` if `s` can be segmented into a space-separated sequence of one or more dictionary words. The same dictionary word may be reused any number of times.

**Constraints:** `1 <= s.length <= 300`; `1 <= wordDict.length <= 1000`; `1 <= wordDict[i].length <= 20`; all strings are lowercase English letters; all dictionary words are unique.

**Examples:**

- `s = "leetcode"`, `wordDict = ["leet","code"]` → `true` (`"leet" + "code"`)
- `s = "catsandog"`, `wordDict = ["cats","dog","sand","and","cat"]` → `false`

## Hints

1. Greedy matching fails (`"cats"` vs `"cat"` in the second example) — you need to consider *all* ways a prefix can be segmented. What subproblem does that suggest?
2. Define `dp[i]` = "can the prefix `s[0..i)` (first `i` characters) be segmented into dictionary words?" What is `dp[0]`?
3. `dp[i]` is true if there exists a split point `j < i` where `dp[j]` is true **and** `s.substring(j, i)` is a dictionary word. Put the dictionary in a `HashSet` for O(1) lookups.
4. Optimization: `j` only needs to range over positions where `i - j` is at most the longest word length in the dictionary.

### Approach / Intuition

This is 1D DP over prefixes of `s`. Let `dp[i]` mean the first `i` characters can be fully segmented. Base case `dp[0] = true` (empty prefix). For each end position `i` from 1 to `n`, look for any split point `j`: if `dp[j]` is true and `s.substring(j, i)` is in the dictionary, then `dp[i]` is true. The answer is `dp[n]`.

Storing `wordDict` in a `HashSet` makes each membership check O(1) (times the cost of building the substring). Iterating `j` from `i-1` downward and capping the word length scanned keeps the inner loop tight.

**Time complexity:** O(n² · L) in the worst case, where `n = s.length()` and `L` is the substring/hash cost — effectively O(n · maxWordLen) split points, each with an O(L) substring + hash.

**Space complexity:** O(n + total dictionary size)

```java
import java.util.HashSet;
import java.util.List;
import java.util.Set;

class Solution {
    public boolean wordBreak(String s, List<String> wordDict) {
        Set<String> dict = new HashSet<>(wordDict);
        int maxLen = 0;
        for (String w : wordDict) {
            maxLen = Math.max(maxLen, w.length());
        }
        int n = s.length();
        boolean[] dp = new boolean[n + 1];
        dp[0] = true; // empty prefix is segmentable
        for (int i = 1; i <= n; i++) {
            for (int j = i - 1; j >= 0 && i - j <= maxLen; j--) {
                if (dp[j] && dict.contains(s.substring(j, i))) {
                    dp[i] = true;
                    break;
                }
            }
        }
        return dp[n];
    }
}
```

**Why this works:** any valid segmentation of `s[0..i)` must end with some dictionary word `s[j..i)`, and the part before it, `s[0..j)`, must itself be segmentable — which is exactly `dp[j]`. The DP checks every possible last word, so no segmentation is missed, and `dp[j]` values are final before `dp[i]` is computed since `j < i`.
