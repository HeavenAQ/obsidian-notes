# 📥 Task Inbox

Use this as the frictionless landing zone for loose tasks before they are assigned to a project, date, or area.

## Inbox

- [ ] Process newly captured tasks every morning #task/inbox

## Capture syntax

- Due date: `📅 YYYY-MM-DD`
- Scheduled/work-on date: `⏳ YYYY-MM-DD`
- Start date: `🛫 YYYY-MM-DD`
- Done date is added by Tasks when completed.
- Priority examples: `🔺` high, `⏫` highest, `🔽` low.
- Reminder example: `(@2026-07-12 09:00)`
